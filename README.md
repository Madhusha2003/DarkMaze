# 🔦 Dark Maze (Flashlight Horror Prototype)

A first-person atmospheric maze exploration game built in **Godot Engine**, focused on learning core 3D game development mechanics like movement, lighting, and gameplay logic.

---

## 🎮 Current Status (MVP Prototype)

### ✔️ Completed Features
- First-person player controller (WASD movement)
- Mouse look (FPS camera control)
- Head bobbing system (walking feel)
- Flashlight system (toggle + smooth fade)
- Dark environment setup
- Fog-based atmosphere
- Physics-based floor and collision system
- Clean project structure setup

---

## 🔧 Core Gameplay Systems

### 🧍 Player System
- Character-based movement
- Camera-relative movement
- Gravity + collision handling
- Head bob effect for realism

### 🔦 Flashlight System
- SpotLight attached to camera
- Toggle ON/OFF (F key)
- Smooth fade transition
- Main visibility mechanic of the game

### 🌑 Environment System
- Fully dark world setup
- Fog-based visibility limitation
- Minimal ambient lighting
- Horror-style atmosphere foundation

---

## 🧱 Current Gameplay Loop

> Walk → Explore dark environment → Use flashlight → Navigate space

(No objective system implemented yet — focus is core mechanics first)

# 🧠 Learning Goals

This project is designed to help understand:

- 3D movement systems
- Camera control mechanics
- Lighting and atmosphere design
- Game state logic (win/lose systems)
- Level design fundamentals
- Clean project architecture

---

# 💡 Design Philosophy

> “Start small, make it feel good early, expand systems gradually.”

Focus:
- gameplay feel > graphics
- simple models > complex assets
- strong mechanics > content volume
